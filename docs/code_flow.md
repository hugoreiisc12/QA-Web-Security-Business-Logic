# Fluxo de Código – Script Demonstrativo para Análise de Lógica de Negócio

1. Usuário navega pela aplicação
2. Frontend realiza request para `/get-user`
3. Script observa a resposta HTTP
4. Dados originais são registrados (log)
5. Simulação client-side é criada
6. QA avalia dependência do frontend
7. Risco é documentado
